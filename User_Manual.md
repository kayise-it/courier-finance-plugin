## 🎯 What is the 08600 Waybill Plugin?

This is a digital system that helps courier companies manage:
- 📋 Delivery slips (waybills)
- 👥 Customer information
- 💰 Price quotes
- 📦 Delivery services
- 🚚 Delivery tracking

Think of it as a digital office for your courier business!

---

## 👥 Who Can Use This System?

There are three types of users:

### 🟢 **Normal Employee** (Basic User)
- Creates waybills
- Views customer information
- Prints delivery slips
- Basic daily tasks

### 🟡 **Manager** (Supervisor)
- Everything an employee can do
- Manages employees
- Reviews and approves waybills
- Generates reports
- Manages services

### 🔴 **Administrator** (Boss)
- Everything a manager can do
- Manages all users
- System settings
- Database management
- Full control

### 📋 Creating a Waybill (Employee)

#### Step 1: Start New Waybill
1. Click **08600 Waybill** → **Create Waybills**
2. You'll see a form with 5 steps

#### Step 2: Customer Information
- **Customer:** Pick from the list (or ask manager to add new customer)
- **From Address:** Where you're picking up
- **To Address:** Where you're delivering
- **Date:** When this delivery happens

#### Step 3: Package Details
- **Item Name:** What you're delivering (e.g., "Books", "Electronics")
- **Quantity:** How many items
- **Weight:** In kilograms
- **Dimensions:** Length × Width × Height
- **Description:** Any special notes

#### Step 4: Review & Save
- Check all information is correct
- Click **Submit** or **Save**
- Print the waybill for the driver

### 👥 Viewing Customers (Employee)
1. Click **08600 Customers** → **All Customers**
2. Use search box to find specific customers
3. Click on customer name to see details
4. You can view but not edit customer information

### 📄 Printing Documents (Employee)
- Look for **Print** or **Download PDF** buttons
- Click to save or print waybills
- Keep copies for your records

### 📊 Managing Waybills (Manager)

#### Reviewing Waybills
1. Go to **08600 Waybill** → **View Waybills**
2. Check status of all deliveries
3. Look for any issues or delays
4. Update status as needed

#### Editing Waybills
1. Find the waybill you want to edit
2. Click **Edit**
3. Make your changes
4. Click **Save**

#### Approving Deliveries
1. Review completed waybills
2. Check delivery confirmation
3. Update status to "Delivered"
4. Add any notes or issues

### 👥 Managing Customers (Manager)

#### Adding New Customers
1. Click **08600 Customers** → **All Customers**
2. Click **Add New**
3. Fill in all required information:
   - **Name:** Full name
   - **Email:** Email address
   - **Phone:** Phone number
   - **Address:** Complete address
   - **City:** City name
   - **Country:** Country
4. Click **Save**

#### Editing Customer Information
1. Find the customer in the list
2. Click **Edit**
3. Update any information
4. Click **Save**

### 📦 Managing Services (Manager)

#### Adding New Services
1. Click **08600 Services**
2. Fill in:
   - **Service Name:** (e.g., "Express Delivery")
   - **Description:** What this service includes
3. Click **Add Service**

#### Examples of Services:
- Same Day Delivery
- Next Day Delivery
- Economy Delivery
- International Shipping
- Special Handling

### 📈 Reports (Manager)

#### Viewing Reports
1. Look for **Reports** or **Analytics** sections
2. Check delivery statistics
3. Monitor customer activity
4. Track service performance

### 👥 Managing Users (Administrator)

#### Creating New Users
1. Go to **Users** → **Add New**
2. Fill in user information:
   - **Username:** Login name
   - **Email:** Email address
   - **Password:** Secure password
   - **Role:** Choose appropriate role
3. Click **Add New User**

#### Setting User Roles
- **Employee:** Basic waybill creation
- **Manager:** Full waybill and customer management
- **Administrator:** Complete system access

#### Managing Permissions
1. Go to **Users** → **All Users**
2. Click **Edit** on any user
3. Change their role as needed
4. Click **Update User**

### ⚙️ System Settings (Administrator)

#### Plugin Settings
1. Look for **Settings** in the 08600 menu
2. Configure:
   - Company information
   - Default services
   - PDF settings
   - Email notifications

#### Database Management
1. **Backup:** Regularly backup your data
2. **Cleanup:** Remove old waybills if needed
3. **Optimize:** Keep database running smoothly

### 🔧 Troubleshooting (Administrator)

#### Common Issues:
1. **Plugin not working:**
   - Check WordPress version compatibility
   - Verify PHP version
   - Check for conflicts with other plugins

2. **PDFs not generating:**
   - Ensure DOMPDF library is installed
   - Check file permissions
   - Verify server memory limits

3. **User access issues:**
   - Check user roles and permissions
   - Verify WordPress user capabilities
   - Clear browser cache

#### Getting Help:
- Check WordPress error logs
- Contact plugin developer
- Review system requirements

## ❓ Common Questions for All Users

### For Employees:
**Q: I can't see the waybill menu?**
- Ask your manager to check your user role
- Make sure you're logged in correctly

**Q: How do I find a customer quickly?**
- Use the search box at the top
- Type part of their name or phone number

**Q: I made a mistake in a waybill, what do I do?**
- Ask your manager to edit it for you
- Don't delete it - managers can fix it

### For Managers:
**Q: How do I approve multiple waybills?**
- Use bulk actions in the waybill list
- Select multiple items and update status

**Q: Can I export customer data?**
- Check for export options in the customer list
- Contact administrator if you need custom exports

**Q: How do I track delivery performance?**
- Use the reports section
- Monitor delivery times and success rates

### For Administrators:
**Q: How do I backup the system?**
- Use WordPress backup plugins
- Export database regularly
- Keep copies of important files

**Q: Can I customize the waybill format?**
- Edit the PDF templates
- Modify CSS styles
- Contact developer for major changes

**Q: How do I update the plugin?**
- Check for updates in WordPress admin
- Test updates on staging site first
- Backup before updating
