Manager Training Manual

This guide shows managers exactly what they can do in the 08600 Solution system, in simple day-to-day steps.

1) Manager Access Overview+

As a Manager, you can usually:

  • Create waybills
  • View and manage waybills
  • Approve waybills
  • Mark waybills as invoiced
  • Manage warehouse assignments
  • Manage deliveries
  • Manage customers (add, edit, view)
  • View dashboard performance numbers

As a Manager, you usually cannot:

  • Access full Settings page (restricted users only)
  • Edit already approved waybills (admin-only flow)
  • See all price-sensitive screens in some cases
  • Delete certain records that require administrator rights
If a button is missing, it is usually a permission setting, not an error.
2) How To Log In And Start+
  1. Log in to WordPress admin.
  2. On the left menu, click 08600 Solution.
  3. You will see the main menu items:
    • Dashboard
    • Create Waybill
    • Manage Waybills
    • Warehouse
    • Customers
    • Routes & Destinations
    • Countries
    • Deliveries
    • Drivers
    • Help
For daily work, start at Dashboard and then move to Manage Waybills.
3) Daily Manager Workflow (Recommended)+

Follow this order every day:

  1. Open Dashboard and check:
    • Waybills created today
    • Waybills in warehouse
    • Deliveries for today
    • Revenue today and this month
  2. Open Manage Waybills:
    • Approve pending waybills
    • Mark completed waybills as invoiced when needed
  3. Open Warehouse:
    • Assign ready waybills to delivery trucks
  4. Open Deliveries:
    • Check delivery status (Scheduled, In Transit, Delivered)
  5. Open Customers:
    • Add or update customer details if needed
4) Dashboard: What You Can Do+

Go to: 08600 Solution > Dashboard

  • View key numbers (today and monthly)
  • Check route and delivery activity
  • Open quick links to create waybills, warehouse, deliveries, customers, and routes
  • View recent waybills

Use dashboard numbers for morning briefing and end-of-day reporting.

5) Create A Waybill+

Go to: 08600 Solution > Create Waybill

  1. Fill customer information.
  2. Fill origin and destination.
  3. Enter package/items information.
  4. Enter dimensions and extra charges (if required).
  5. Review all details.
  6. Submit.

Best practice

  • Double-check destination and contact details before submit.
  • Confirm package counts and dimensions are correct.
6) Manage Waybills (Most Important Page)+

Go to: 08600 Solution > Manage Waybills

  • Search for waybills
  • Sort and filter waybills
  • Open a waybill to view full details
  • Delete selected waybills (where permission allows)
  • Export selected waybills to PDF
  • Download waybill summary PDF

Approve A Waybill

  1. Open the waybill from the list.
  2. Click Approve (if visible).
  3. Confirm status changed.

Mark A Waybill As Invoiced

  1. Open an approved waybill.
  2. Click Invoice or Mark as Invoiced (label may vary).
  3. Confirm invoicing status was saved.
If approval/invoice buttons are not visible, check your role permissions.
7) Warehouse: Assign Waybills To Deliveries+

Go to: 08600 Solution > Warehouse

  • View waybills in warehouse by status
  • Filter by status/country
  • Select multiple waybills
  • Assign selected waybills to a scheduled delivery
  • Add a new delivery from the warehouse modal

Steps:

  1. Filter to the correct country/status.
  2. Tick waybills ready to dispatch.
  3. Choose the target delivery.
  4. Click assign.
  5. Verify assignment success message.
8) Deliveries: Track Truck Progress+

Go to: 08600 Solution > Deliveries

  • View all deliveries
  • Create delivery truck entries (manager/admin flow)
  • Open delivery details
  • Update delivery status:
    • Scheduled
    • In Transit
    • Delivered
Daily target: Ensure every active truck has the correct status before close of business.
9) Customers: Add And Maintain Records+

Go to: 08600 Solution > Customers

  • Add new customers
  • View and edit customer details
  • View customer waybill history
  • Download customer-related PDFs
  • Upload customers via CSV (if format is correct)
Customer delete may be restricted to administrator users.
10) Routes, Countries, Drivers+

Go to:

  • Routes & Destinations
  • Countries
  • Drivers

Depending on permissions, you may be able to:

  • Add/edit routes
  • Activate/deactivate countries
  • Add/edit drivers

Note: In some environments, these actions are limited to specific admin-level users.

11) PDF And Reporting Tasks+

Managers can commonly generate:

  • Single waybill invoice PDF
  • Waybill summary PDF
  • Bulk waybill PDF export
  • Customer-related PDF exports

Where to generate:

  • Manage Waybills (single and bulk exports)
  • Customers (customer and bulk exports)
  • Waybill detail page (invoice PDF)
12) Troubleshooting (Easy Checks)+

If something is not working:

  1. Refresh the page once.
  2. Confirm you are on the correct menu page.
  3. Check required fields are completed.
  4. Check your permissions (missing button often means restricted access).
  5. Log out and back in.
  6. If still blocked, send screenshot + waybill/customer ID to admin support.
13) Manager Do/Do Not Checklist+

Do

  • Approve waybills quickly to keep operations moving.
  • Keep delivery statuses updated through the day.
  • Verify customer details before creating shipments.
  • Export reports/PDFs for finance and operations review.

Do Not

  • Assume missing buttons are bugs (usually role restrictions).
  • Edit records after final approval without admin direction.
  • Leave delivery statuses outdated at day end.
14) Quick Menu Cheat Sheet+
  • Dashboard -> Daily numbers and quick actions
  • Create Waybill -> New shipment record
  • Manage Waybills -> Search, approve, invoice, export
  • Warehouse -> Assign waybills to deliveries
  • Deliveries -> Track and update truck status
  • Customers -> Add/edit customers and export customer PDFs
  • Help -> Reference links (if configured)
End of manual.
